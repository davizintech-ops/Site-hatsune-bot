Como Instalar o Hatsune Bot

Siga os passos abaixo:

1. Clone o repositório:
git clone https://github.com/davizintech-ops/hatsune-bot.git
2. Entre na pasta:
cd hatsune-bot
3. Instale as dependências:
npm install
4. Configure o ambiente:
cp .env.example .env
5. Inicie o bot:
npm start

Pronto! Seu Hatsune Bot estará rodando 🚀
